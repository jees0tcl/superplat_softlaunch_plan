🎨 아트실 싱크업 - 매주 수요일의 핵심 프로세스

📅 매주 수요일 - 아트와 개발의 완벽한 동기화

목적:
  • 아트팀과 개발팀 간의 실시간 협업 체계 구축
  • 새로운 콘텐츠와 애셋의 즉각적인 통합
  • 예측 가능한 업데이트 사이클 확립
  • 매주 목요일 정기 업데이트의 기준점 제공

⏰ 수요일 타임라인

화요일까지: 아트팀 작업 완료 및 제출
10:00-12:00: 아트 리뷰 세션 (품질 검토, 피드백)
13:00-15:00: 데이터 싱크업 (메타데이터 입력, 서버 업데이트)
15:00-17:00: 통합 테스트 및 검증
목요일: 내부 배포 및 전사 테스트 → 정기 업데이트 기준
🚀 핵심 가치:

이 프로세스는 단순한 회의가 아닙니다. 매주 수요일의 싱크업이 목요일 업데이트의 기준이 되어, 주 단위로 예측 가능한 콘텐츠 업데이트 사이클을 만듭니다. 아트팀은 창작에 집중하고, 개발팀은 계획적으로 통합하며, 전 직원은 매주 새로운 콘텐츠를 경험할 수 있습니다.

포함 작업:
  • 신규 애셋 제작 및 통합
  • 기존 애셋 폴리싱 및 최적화
  • 성능 최적화 작업
  • UI/UX 개선사항 반영

핵심 기술 과제

🔧 캐릭터 시스템 통합

담당: 양지석
목표: 9월 3주차 완료
현재 문제: 플레이어/NPC/커스터마이징 각각 다른 클래스 사용
기대 효과: 개발 속도 2배 이상 향상, 버그 감소

⚡ 성능 최적화

담당: 조상원
뮤터블 시스템: 드로우콜 10→3개 감소
프레임레이트: 30→60fps 향상
추가 작업: 메모리 누수 해결, 로딩 시간 30% 단축
효과: 저사양 디바이스에서도 많은 플레이어 표현 가능

🖥️ 서버 안정화

담당: 노남권
목표 지표:
  • 필드당 동시접속 80명 안정 처리
  • 서버군당 동시접속 1,000명 수용
  • 평균 응답시간 1초 이내
  • 서버 접속 실패시 예외 처리 필수
  • 분산 서버 구조 구축
일정: 9월 3주 진단 → 4주 계획 → 10월 실행

🎮 이벤트 시스템

담당: 신정효, 노남권, 김진형
11월 목표: 기반 시스템 구축
12월 목표:
  • 퀘스트 시스템 완성
  • 포토카드 선물박스 찾기
